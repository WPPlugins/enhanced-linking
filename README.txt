This plugin enhances the Insert/Edit Link dialogue by letting users select and find additional content from their blog and external web.

== Description ==

Once Enhanced Linking is installed, users will notice two new options: ability to add links from related articles and ability to search the web right from the window. This will let users find relevant fresh content from other bloggers and allow them to find anything on the web in a quick and easy fashion.

== Installation ==

1. Unzip the file, copy `enhanced_linking` directory to `wp-content/plugins` on your server.
2. Inside WordPress dashboard there's the `Plugins` menu, click on it. Then click on `Activate` link under Zemanta plug-in.
3. Start writing a new post. See the suggested content after you write a few sentences!
